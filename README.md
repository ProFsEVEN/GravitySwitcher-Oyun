# GravitySwitcher

Bu, .NET 9.0 ile geliştirilmiş bir 'GravitySwitcher' projesidir.

## Kurulum

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

1.  Bu depoyu klonlayın:
    ```bash
    git clone https://github.com/ProFsEVEN/GravitySwitcher-Oyun.git
    ```
2.  Proje dizinine gidin:
    ```bash
    cd web2oyun
    ```
3.  Bağımlılıkları yükleyin ve projeyi çalıştırın:
    ```bash
    dotnet run
    ```

## Özellikler

- .NET 9.0 Framework
- Web tabanlı oyun mekanikleri
